Gerencie os clientes de sua pizzaria

Além de um projeto de estudos, o "Humm Pizzaiolo" tem como intuíto se tornar um software competitivo, com base em atender os processos internos e rotinas de comércio de uma pizzaria.

"...se tornar um software..."?!

Sim! Pelo momento, o projeto encontra-se em fase de levantamento dos requisitos que envolvem este tipo de negócio. Pretenção inicial: Interface Web

"...software competitivo..."?

Inicialmente o "Humm Pizzaiolo" apresentará propostas básicas de um sistema comum, com funcionalidades gerais como: Cadastro de Clientes, Cadastro de Produtos e Cadastro de Pedidos.

O plano futuro para este software é que ele se torne referência de mercado no segmento, trazendo melhorias das ideias iniciais, implementações de funcionalidades administrativas e financeiras, possibilitando compras online e facilidades de acesso.

Powered by Java: OpenSource = Baixo custo

O desenvolvimento deste produto está baseado inicialmente em Java, linguagem de programação que os autores do projeto estão aprendendo no curso Tecnologia em Análise e Desenvolvimento de Sistemas, realizado na instuição SENAC São Paulo - Campus Santo Amaro, com orientação dos professores: William Honda e Alessandra Aguiar - especialistas em software.

Evoluir

A linguagem Java possibilita o desenvolvimento de softwares livre de custos, além de ser multiplataforma, expansível e ramificada.

Como assim?

Além de tudo, muitas linguagens novas da atualidade estão evoluindo com base na plataforma da Máquina Virtual Java, o quê facilitará a migração futura, se necessária.
